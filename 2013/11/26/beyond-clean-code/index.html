Beyond Clean Code

If code delivers good business value, then it’s good code. That doesn’t mean we can’t improve it. It just means that it is doing what it needs to!

Source: ircmaxell’s blog

I really like what Anthony Ferrara said. Sometimes it’s easy to get lost in the debate how to structure your code or which programming pattern to use. But at the end of the day, it’s business value that matters.
